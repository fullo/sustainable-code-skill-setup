| [03-page-emissions.md](03-page-emissions.md) | Estimate CO2 emissions for a website |

## Prerequisites

- Claude Code (or any agent supporting agentskills.io)
- Skills installed in `.claude/skills/` ([installation guide](../README.md#installation))
- MCP tools (optional, for real-time calculations)
